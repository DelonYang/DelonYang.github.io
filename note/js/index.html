Number

Number包括了整型、单精度、双精度； 计算时候容易出现脑残问题：0.1 + 0.2 = 0.30000000000000004.

常用函数：

// 字符串 -> 整数
parseInt('100', 10)

// 字符串 -> 浮点数
parseFloat('1.5')
